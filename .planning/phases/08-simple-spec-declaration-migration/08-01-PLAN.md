
Construction (leaf-to-root, innermost first):
1. Build the inner lambda: `ParenthesizedLambdaExpression` with a single typed parameter `(modelTypeName parameterName)` and `bodyExpression` as the expression body
2. Build `Spec.Build(innerLambda)`: `InvocationExpression` with `MemberAccessExpression(IdentifierName("Spec"), IdentifierName("Build"))` and the lambda as argument
3. Chain `.WhenTrue("(escapedOriginalExpression) == true")`: `InvocationExpression` with `MemberAccessExpression` on the prior result, with a string literal argument using `LiteralExpression(SyntaxKind.StringLiteralExpression, Literal($"({escapedOriginalExpression}) == true"))`
4. Chain `.WhenFalse("(escapedOriginalExpression) == false")`: Same pattern as WhenTrue
5. Chain `.Create()`: `InvocationExpression` with `MemberAccessExpression`, empty `ArgumentList()`

String interpolation IS acceptable for the WhenTrue/WhenFalse string literal values per project decision -- these are runtime strings, not source code structure.
